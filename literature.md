## Бесплатно онлайн

| Книга | Авторы | Фокус | Ссылка |
|---|---|---|---|
| Deep Learning | Goodfellow, Bengio, Courville | Теория DL, библия | [deeplearningbook.org](https://deeplearningbook.org) |
| Dive into Deep Learning | Zhang et al. | Практика, весь код на PyTorch | [d2l.ai](https://d2l.ai) |
| Mathematics for Machine Learning | Deisenroth et al. | Математика под ML с нуля | [mml-book.github.io](https://mml-book.github.io) |
| Pattern Recognition and Machine Learning | Bishop | Фундаментальная теория ML | [PDF на сайте автора](https://www.microsoft.com/en-us/research/publication/pattern-recognition-machine-learning/) |

## Порядок чтения

**Если только начинаешь:**
```
Géron → d2l.ai → Goodfellow (по мере надобности)
```

**Если есть матбаза:**
```
MML → Bishop → Goodfellow
```
