# SciPy — Learning Roadmap

> **Цель:** научная математика поверх NumPy — статистика, оптимизация, обработка сигналов, линейная алгебра.

Прогресс: `[ ]` не начато · `[~]` в процессе · `[x]` изучено

> **Примечание:** SciPy — модульная библиотека. Изучайте только те разделы, которые нужны для ваших задач. Уровни 1–3 актуальны почти всегда.

---

## Уровень 1 — Статистика (`scipy.stats`) ⭐

- [ ] Непрерывные распределения: `norm`, `t`, `chi2`, `f`, `expon`, `uniform`
- [ ] Дискретные распределения: `binom`, `poisson`
- [ ] `rvs` — генерация выборки, `pdf`/`pmf` — плотность, `cdf` — функция распределения
- [ ] Описательная статистика: `describe`, `skew`, `kurtosis`
- [ ] Проверка нормальности: `shapiro`, `normaltest`
- [ ] Параметрические тесты: `ttest_1samp`, `ttest_ind`, `ttest_rel`
- [ ] Непараметрические тесты: `mannwhitneyu`, `wilcoxon`, `ks_2samp`
- [ ] Корреляция: `pearsonr`, `spearmanr`, `kendalltau`
- [ ] ANOVA: `f_oneway`
- [ ] Подбор распределения к данным: `fit`

---

## Уровень 2 — Оптимизация (`scipy.optimize`) ⭐

- [ ] Минимизация функции: `minimize` — метод, опции, ограничения ⭐
- [ ] Методы: `'Nelder-Mead'`, `'BFGS'`, `'L-BFGS-B'`, `'SLSQP'`
- [ ] Ограничения: `bounds`, `constraints`
- [ ] Поиск корней: `brentq`, `fsolve`
- [ ] Подбор кривой: `curve_fit` ⭐ — подгонка произвольной функции к данным
- [ ] Линейное программирование: `linprog`
- [ ] Глобальная оптимизация: `differential_evolution`, `dual_annealing`

---

## Уровень 3 — Интерполяция (`scipy.interpolate`)

- [ ] `interp1d` — одномерная интерполяция
- [ ] Виды: `'linear'`, `'nearest'`, `'cubic'`, `'quadratic'`
- [ ] `CubicSpline` — кубический сплайн
- [ ] `griddata` — интерполяция на неравномерной сетке (2D)
- [ ] `RectBivariateSpline` — 2D интерполяция

---

## Уровень 4 — Линейная алгебра (`scipy.linalg`)

- [ ] Разложения: `lu`, `qr`, `svd`, `cholesky`
- [ ] Собственные значения: `eig`, `eigh` (для симметричных матриц)
- [ ] Решение систем: `solve`, `lstsq`
- [ ] Псевдообратная матрица: `pinv`
- [ ] Отличие `scipy.linalg` от `numpy.linalg` — когда что использовать

---

## Уровень 5 — Обработка сигналов (`scipy.signal`)

- [ ] Свёртка и корреляция: `convolve`, `correlate`
- [ ] Фильтры: `butter`, `sosfilt` — проектирование и применение
- [ ] Типы фильтров: lowpass, highpass, bandpass, bandstop
- [ ] Спектр: `periodogram`, `welch`
- [ ] Нахождение пиков: `find_peaks` и его параметры

---

## Уровень 6 — Разреженные матрицы (`scipy.sparse`)

- [ ] Зачем разреженные матрицы (NLP, рекомендательные системы)
- [ ] Форматы: `csr_matrix`, `csc_matrix`, `coo_matrix`
- [ ] Создание, индексация, преобразование в dense
- [ ] `sparse.linalg`: `spsolve`, `svds` — SVD для разреженных матриц
- [ ] Совместимость с sklearn (многие алгоритмы принимают sparse)

---

## Уровень 7 — Интегрирование и дифференциальные уравнения

- [ ] Численное интегрирование: `quad` (одномерное), `dblquad` (двойное)
- [ ] Решение ОДУ: `solve_ivp` ⭐ — современный интерфейс
- [ ] Методы: `'RK45'`, `'DOP853'`, `'Radau'`
- [ ] Анализ результата: события, плотный вывод

---
