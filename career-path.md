# Уровень и дальнейший путь

---

## Что даёт текущий план

После прохождения всех блоков (00–07) + литературы — уверенный **Middle ML Engineer**.

### Что умеешь

- Брать сырые данные, чистить, исследовать, строить модели
- Работать с CV и NLP задачами через HuggingFace
- Дообучать готовые модели (fine-tuning)
- Упаковывать модель в API и деплоить
- Вести эксперименты профессионально (MLflow, DVC, Docker)

### Чего пока не хватает для Senior / Research

- Писать архитектуры с нуля под нестандартные задачи
- Глубокая математика (теория оптимизации, байесовский вывод)
- Распределённое обучение на кластерах
- LLM-специфика: RLHF, квантизация, vLLM, inference optimization

---

## Куда двигаться дальше

### Блок A — Углублённая математика
- Теория оптимизации (выпуклая оптимизация, Adam изнутри)
- Байесовская статистика и вероятностные модели
- Теория информации (entropy, KL-divergence)

### Блок B — LLM и современный NLP
- Fine-tuning LLM: LoRA, QLoRA, PEFT
- Квантизация моделей: GPTQ, bitsandbytes
- Inference: vLLM, TGI, speculative decoding
- RAG (Retrieval-Augmented Generation)
- Agents: LangChain, LlamaIndex

### Блок C — Продвинутый CV
- Object detection: YOLO, DETR
- Segmentation: SAM (Segment Anything)
- Generative models: Stable Diffusion, ControlNet
- Multimodal: CLIP, LLaVA

### Блок D — Scale и Production
- Распределённое обучение: DeepSpeed, FSDP
- Инференс на GPU: TensorRT, ONNX Runtime
- Мониторинг моделей в production: Evidently, Grafana
- Feature Store: Feast
- Оркестрация: Airflow, Prefect
